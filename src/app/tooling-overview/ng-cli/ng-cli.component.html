> Angular CLI (Command Line Interface)

  • The Angular CLI is an official Angular package that allows for easily bootstraping an application
  • Abstracts away Webpack and allows for "app ejection" -- which is short for give me my config and set up my npm scripts
  • Has support for service workers and server side rendering -- undoubtedly each come with associated headaches.
  • Not ideal for creating consumable libraries. Designed more for application development. Look into ng-packagr for more info.
