  • There is no more $digest cycle. *hoo-rah I hear the crowd cheer*
  • Angular runs in a Zone. This is an area of the DOM in which async methods are intercepted and managed by Zone.js.
  • This means that change detection is ran only when asynchronous code could have altered our data.
  • One can set the component meta-data such that change detection for this component is only ran OnPush
  • This will also effect components further down the component tree.
