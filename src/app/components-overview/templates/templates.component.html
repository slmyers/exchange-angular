  • attributes wrapped in [] denote property bindings while those wrapped in () represent event bindings.
  • directives marked with * are called structural directives and will make changes to the DOM structure.
  • directives without this prefix are called attribute directives and are used to change the appearance or behavior of a DOM element.
  • The | operator feeds input through a pipe. Analogous to AngularJS filter.
  • The async pipe will resolve a promise or manage an Observable such that the value is available in the template.
  • ng-template is a wrapper around native DOM template. The # attribute is akin to an id.
